### Study Materials: Video Resources
* [Hub, Switch & Router](https://www.youtube.com/watch?v=1z0ULvg_pW8&t=1s)
    - **hub**: signal replicator - it replicates the signals to all the network devices connected on an internal network
    - **switch**: works in a similar way while keeps the MAC address of each device connected to it, so it only sends the signal to its intended device destination
    - **router**: read the IP Address of a data packet from a device and foward the data out on the internet until it reaches to the intended network device.

* [Default Gateway](https://www.youtube.com/watch?v=pCcJFdYNamc)
    - A **default gateway** it's the standard choice when data exit from within network to comunicate with devices on another network.
